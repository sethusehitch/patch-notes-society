# Patch Notes for Society v0.1

Patch Notes for Society is now public as a v0.1 civic systems project.

The site includes stable public pages for ten issue lanes:

- #001 Prisons
- #002 Housing / Homelessness
- #003 Healthcare Affordability
- #004 Education
- #005 Addiction / Mental Health
- #006 Money in Politics / Corruption
- #007 Immigration
- #008 Climate / Energy Transition
- #009 Cost of Living / Inequality
- #010 Gun Violence / Public Safety

The goal is not to claim that first drafts solve these systems. The goal is to make public problems easier to inspect, criticize, test, fund, govern, and improve.

## What Is Live

- Public site: https://sethusehitch.github.io/patch-notes-society/
- Issue pages for #001-#010
- Sitemap and robots.txt
- Social preview metadata
- Public GitHub feedback forms for critique, source corrections, implementation risks, and general project feedback
